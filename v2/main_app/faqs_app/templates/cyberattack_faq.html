General

First, make sure you are using chrome to access this website. If the site is blocked by your school, then you may have to access the website in the virtual machine. If you are trying to do this, make sure you are using chrome to access this website.

Your host ID and password can be found in three places.

  1. It can be found at the start of the command prompt that Shinobot is running on.
  2. It can be found by hovering over the application in the toolbar.
  3. It can be found where it says shinobotps on the application at the top.
